MADCIX

MADEIRA CLOUD INTERNET EXCHANGE

O MAD-CIX tem como objetivo disponibilizar uma plataforma para organizações nacionais e internacionais onde o tráfego IP é comutado a alta velocidade, promovendo encaminhamento do tráfego IP com baixa latência entre os seus membros.

A adesão como membro do MAD-CIX é aberta a todas as organizações que cumpram os requisitos técnicos e legais, facilitando uma plataforma de troca de tráfego IP a todos os seus membros.

A troca de tráfego no MAD-CIX é realizada através de acordos de peering entre as partes locais e internacionais sendo usado o BGP (Border Gateway Protocol).

Aumentam-se as interligações na Madeira de várias redes de IP, reduz-se o excesso de utilização de recursos internacionais, menos custos e melhor qualidade para trânsito de pacotes IP com ligações de origem e destino na Madeira para o Mundo.
